## ESLint と Prettier の基本概念

1. ESLint

   - コードの品質チェックツール
   - 潜在的なバグやエラーを見つける
   - コーディング規約を強制する
   - 例：

     - 未使用の変数の検出
     - 適切でない構文の指摘
     - ベストプラクティスの推奨

2. Prettier

   - コードフォーマッター
   - コードの見た目を整える
   - 例：

     - インデントの統一
     - 改行の位置
     - クォーテーションの統一

   - コマンド
     - docker-compose exec app npm install --save-dev eslint
